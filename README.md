# Life Manager (Web-first) — UI Starter

Стартовый проект на Flutter Web для командного проекта **Life Manager** (трекер привычек + финансы + планировщик).
Сейчас здесь **верстка и UI**, без реального бэкенда. Все экраны уже есть, чтобы показать преподавателю структуру будущего функционала.

## Быстрый старт
1) Установите Flutter (stable) и включите web:
```bash
flutter --version
flutter config --enable-web
```

2) В корне проекта:
```bash
flutter pub get
flutter run -d chrome
```

## Страницы
- Dashboard (главная)
- Habits (привычки)
- Finance (доход/расход)
- Planner (задачи / расписание)
- Profile
- Settings (тема/язык)

## Командные коммиты (пример)
- Бота: базовый каркас, навигация, тема
- Ерке: страница Habits (карточки, макеты форм)
- Ерасыл: Finance (карточки, таблицы, графики позже)
- Асхат: Planner + Profile (макеты)
